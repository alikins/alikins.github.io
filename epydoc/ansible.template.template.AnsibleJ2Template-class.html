                             AnsibleJ2Template

A helper class, which prevents Jinja2 from running _jinja2_vars through dict(). Without this, {% include %} and similar will create new contexts unlike the special one created in template_from_file. This ensures they are all alike, except for potential locals.

Method Details [hide private]

new_context(self, vars=None, shared=False, locals=None)

source code 

Create a new :class:`Context` for this template. The vars provided will be passed to the template. Per default the globals are added to the context. If shared is set to `True` the data is passed as it to the context without adding the globals.

`locals` can be a dict of local variables for internal usage.
