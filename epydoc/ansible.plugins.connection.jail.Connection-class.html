_prefix_login_path(self, remote_path)

source code 

Make sure that we put files into a standard path

If a path is relative, then we need to choose where to put it. ssh chooses $HOME but we aren't guaranteed that a home dir will exist in any given chroot. So for now we're choosing "/" instead. This also happens to be the former default.

Can revisit using $HOME instead if it's a problem
