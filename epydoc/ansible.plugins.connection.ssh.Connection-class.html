source code 

Takes a command array and scans it for ControlPersist and ControlPath settings and returns two booleans indicating whether either was found. This could be smarter, e.g. returning false if ControlPersist is 'no', but for now we do it simple way.

_send_initial_data(self, fh, in_data)

source code 

Writes initial data to the stdin filehandle of the subprocess and closes it. (The handle must be closed; otherwise, for example, "sftp -b -" will just hang forever waiting for more commands.)

_examine_output(self, source, state, chunk, sudoable)

source code 

Takes a string, extracts complete lines from it, tests to see if they are a prompt, error message, etc., and sets appropriate flags in self. Prompt and success lines are removed.

Returns the processed (i.e. possibly-edited) output and the unprocessed remainder (to be processed with the next chunk) as strings.
