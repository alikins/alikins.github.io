 52      optional.  The format string is in the style of printf.  This can be used

 55      Negative numbers are not supported. 
 56   
 57      Some examples: 
 58

 60        5-8 -> ["5", "6", "7", "8"] 
 61        2-10/2 -> ["2", "4", "6", "8", "10"] 
 62        4:host%02d -> ["host01","host02","host03","host04"] 
 63   
 64      The standard Ansible key-value form is accepted as well.  For example: 
 65   
 66        start=5 end=11 stride=2 format=0x%02x -> ["0x05","0x07","0x09","0x0a"] 
 67   
 68      This format takes an alternate form of "end" called "count", which counts 
 69      some number from the starting value.  For example: 
 70
